- Create new project
- Get your 6 API credentials
- Copy them (you'll need them)

### Command #3: Deploy to Vercel
Go to: https://vercel.com/signup
- Sign in with GitHub
- Import "reactor" repository
- Add Firebase environment variables
- Click Deploy

**Result: You get a live URL like `https://reactor-xyz.vercel.app`** ✅
